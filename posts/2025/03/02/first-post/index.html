最初の投稿です。
ここでは、サイトの構築環境について説明します。

本サイトは静的サイトジェネレーター Hugo と GitHub Pages を用いて構築されています。
テーマには hugo-coder を採用しました。

ソースコードはprivateリポジトリで管理しています。
コードをプッシュすると、GitHub Actions により自動的にビルドプロセスが実行され、生成された静的ファイルがpublicリポジトリの hukisanagi.github.io にデプロイされる仕組みになっています。
